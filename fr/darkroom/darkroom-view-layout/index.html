                  disposition de la vue chambre noire

                  🔗panneau gauche

                  De haut en bas :

                  navigation
                  Naviguer et zoomer la vue centrale.
                  instantanés
                  Prendre et afficher des instantanés pour les comparer avec le développement actuel.
                  gestion des clones
                  Afficher et gérer les clones
                  pipette de couleur
                  Sélectionner et afficher les informations de couleur provenant de parties de l’image.
                  mots-clés
                  Gérer les mots-clés.
                  informations de l’image
                  Afficher des informations de l’image actuelle
                  gestion des masques
                  Afficher et modifier les formes dessinées
                  exporter
                  Exportez les images sélectionnées vers des fichiers locaux ou des services externes.

                  🔗panneau droit

                  De haut en bas :

                  scopes
                  A graphical depiction of the image’s light levels and colors. This module can be moved to the left-hand panel if desired (see preferences > miscellaneous > position of the scopes module).
                  groupes de modules
                  Sélectionnez des groupes de modules (si activé)
                  search module
                  Search for a module (if enabled).
                  modules de traitement
                  Les modules utilisés pour traiter une image.
                  ordre des modules
                  Choisir l’ordre dans lequel les modules de traitement sont exécutés dans le pipeline graphique.

                  🔗panneau inférieur

                  De gauche à droite :

                  préréglages
                  Menu d’accès rapide aux préréglages des modules. Vous pouvez gérer le contenu de ce menu en sélectionnant “gérer la liste des préréglages…”
                  styles
                  Menu d’accès rapide aux styles
                  second darkroom window
                  For multi-monitor setup, allows the user to display a second image on another screen.
                  détection de focus
                  Activer/désactiver le mode détection de focus.
                  évaluation des couleurs
                  Activer/désactiver la vue évaluation des couleurs ISO12646.
                  icône-zones-surexposées-RAW avertissement surexposé RAW
                  Activer/désactiver les indicateurs de zones surexposées RAW (clic-droit pour les options)
                  [avertissement d’écrêtage]../module-reference/utility-modules/darkroom/clipping.md)
                  Activer/désactiver les avertissements d’écrêtage (clic-droit pour les options)
                  épreuvage écran
                  Activer/désactiver les superpositions de l’épreuvage écran (clic-droit pour les options).
                  vérification du gamut
                  Activer/désactiver la vérification du gamut (clic-droit pour les options)
                  guides & overlays
                  Left-click to switch global guide overlays on/off and right-click to change the guide settings, including the color of all on-image drawing (masks, crop guides etc.)

                  You can also enable the filmstrip module at the bottom of the screen to allow you select and interact with the currently selected collection in the lighttable view.
